# eTIMS Invoice Guide (Kenya)

A simple, interactive web guide for creating compliant eTIMS invoices in Kenya using official KRA exchange rates.

## Features

- **Auto-calculating invoice inputs** - Enter USD amount and KRA rate, get instant KES conversion
- **Persistent storage** - Your inputs save automatically in browser localStorage
- **One-click copy actions** - Copy exchange rate note, KES amount, or buyer details
- **Step-by-step workflow** - Clear instructions for monthly invoicing and credit note corrections
- **FAQ & troubleshooting** - Common questions about downloads, rates, and mistakes
- **Light/dark mode** - Auto-detects system preference with manual toggle

## Usage

1. Visit the page
2. Fill in your invoice inputs (buyer details, USD amount, KRA rate)
3. Copy the auto-generated note and KES amount
4. Follow the monthly steps to create your eTIMS invoice
5. Your settings persist automatically for next time

## Tech Stack

- Pure HTML/CSS/JavaScript (no dependencies)
